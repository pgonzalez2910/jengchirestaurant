Physical Count Price Grid

Simple grid for all Physical Count items. Choose the Physical Count unit, invoice pack type, units inside the pack, and LB per saved unit. The price is calculated from the latest invoice and auto-saved back to the dashboard.
Use this grid as the only Admin Setup Review. Add a new invoice, click Refresh, then choose how the item should be counted on the Physical Count page. Turn Auto Save ON to save unit, price, and LB/package immediately after each change. This also updates the current Physical Count row for the selected inventory date. Produce normally uses LB. Flour normally uses BAG. A case with 12 inside can use Units Inside Invoice Pack = 12 so the case price is divided automatically.
