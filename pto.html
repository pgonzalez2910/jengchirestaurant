PTO Hours and PTO Wage Dashboard

Consolidated online view for plan years 2025 and 2026. This dashboard shows PTO hours earned, hourly rate, PTO wage value, and employee-level totals across both years. It is ready to connect to a database-backed API for live reporting.
Source workbook: PTO BALANCE REPORT.xlsx
Period covered: 2025 and 2026
Compliance note: the source file does not include a unique employee ID. For production use, map each row to your payroll or HRIS employee ID to avoid ambiguity for duplicate names or job changes.

Reporting Scope

Employee detail, year totals, cross-year totals, and searchable records in one page.

Data Model

Employees + yearly PTO balances. The SQL script creates a normalized structure and views for the web layer.
